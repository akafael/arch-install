# Arch Install

Install script for [Arch Linux](https://wiki.archlinux.org/index.php/Arch_Linux) using Makefile.

### Install through Makefile script

```
pacman -Syu
pacman -S git
git clone https://github.com/akafael/arch-install.git
cd arch-install
make
```
